# Customer-Churn-Analysis
Analyzed customer data to identify churn patterns Performed data cleaning and exploratory analysis
Tools Used:

Excel / Power BI
Basic Data Analysis

Key Analysis Performed:

Data cleaning and preprocessing
Customer churn distribution
Tenure and monthly charges analysis
Visualization using dashboard

🔍 Key Insights:

Customers with low tenure have higher churn
Higher monthly charges increase churn probability
Certain customer segments are more likely to churn

💡 Business Recommendations:

Improve onboarding for new customers
Provide discounts or offers for high-risk customers
Focus on customer retention strategies
